## <p align="center">🚀 Featured Projects</p>

### 🛒 Voltara Solar | Full-stack e-commerce platform
A complete online store for a concept solar brand, with product browsing and checkout, customer accounts, a REST API secured with JWT and bcrypt, and a full admin catalog system.
`Node.js` `Express.js` `MongoDB` `Mongoose` `REST API` `JWT` `EJS` `Bootstrap` `jQuery`

### 🤖 AI Exam Practice Platform
An AI-powered platform where students attempt past-paper questions and get instant marking against the official mark schemes, with topic-level weakness tracking. Built solo, front to back.
`React` `Vite` `FastAPI` `Python` `Claude API` `PostgreSQL` `Supabase`

### 🌊 AidSync | Disaster aid platform
A team-built MVP that connects flood-affected families, NGOs, and donors, with four ML models behind a FastAPI backend for demand prediction, donor matching, and scam detection.
`Python` `FastAPI` `PyQt5` `MS SQL Server` `Machine Learning`

### 💣 DSA Minesweeper | Algorithmic game engine
Minesweeper rebuilt in C++ and SFML as a data-structures showcase, using a 2D grid, BFS, linked lists, and heaps.
`C++` `SFML` `Data Structures` `Algorithms`
